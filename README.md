# Materiales Didácticos UTEQ

Repositorio de materiales didácticos para asignaturas de Ingeniería, organizado y presentado mediante un sitio web estático con Jekyll.

## Objetivo

Este repositorio tiene como propósito organizar, desarrollar y compartir material didáctico para la enseñanza de diversas asignaturas de Ingeniería a nivel universitario. Está diseñado tanto para uso personal como para futura colaboración con pares académicos.

## Asignaturas Incluidas

### Física Moderna
El alumno describirá los fenómenos fundamentales de la física moderna y aplicará sus principios para comprender el comportamiento cuántico de la materia a nivel atómico utilizando herramientas computacionales y de inteligencia artificial como apoyo.

### Ingeniería de Materiales
Estudio de las propiedades, comportamiento y aplicaciones de diversos materiales en ingeniería, abarcando metales, cerámicos, polímeros, compuestos y nanomateriales, así como sus procesos de selección y manufactura.

## Estructura del Sitio

El sitio utiliza [Jekyll](https://jekyllrb.com/) y el tema [Just the Docs](https://just-the-docs.com/).

- `_cursos/`: Contiene el material de cada asignatura.
- `assets/`: Archivos estáticos (imágenes, documentos PDF, estilos).
- `_config.yml`: Configuración principal del sitio.

## Desarrollo Local

### Requisitos previos
1. Ruby (versión 2.5.0 o superior)
2. RubyGems
3. Bundler

### Instalación y Ejecución

1. Instala las dependencias:
   ```bash
   bundle install
   ```

2. Ejecuta el servidor local:
   ```bash
   bundle exec jekyll serve
   ```

3. Accede al sitio en `http://localhost:4000/PreparacionCursosJ/`
